# 服務項目整合說明

## 🎯 整合完成

已成功將 `服務項目.json` 的數據與卡片組件整合，現在服務頁面會自動顯示所有服務項目。

## 📁 文件結構

### 修改的文件
1. **`src/components/card.vue`** - 卡片組件
2. **`src/views/servicesViews.vue`** - 服務頁面
3. **`src/assets/css/services.css`** - 服務頁面樣式

### JSON 數據文件
- **`src/assets/json/服務項目.json`** - 服務數據來源

## 🔧 組件屬性

### ProductCard 組件現在支持的屬性：
```vue
<ProductCard
  :image="product.image"    // 圖片路徑
  :name="product.name"      // 服務名稱  
  :time="product.time"      // 服務時間
  :colors="product.colors"  // 可選：顏色選項
  :originalPrice="product.originalPrice"  // 可選：原價
  :salePrice="product.salePrice"          // 可選：優惠價
/>
```

## 📋 JSON 數據映射

### 原始 JSON 結構：
```json
{
  "孕期課程": [
    { "課程名稱": "孕婦按摩", "時間": "80min" }
  ]
}
```

### 映射到卡片屬性：
- `課程名稱` → `name` 屬性
- `時間` → `time` 屬性
- 分類名稱 → 用於圖片選擇和分組顯示

## 🖼️ 圖片映射策略

目前使用的圖片映射：
- **孕期課程**: `/img/服務項目/產康圖2.jpg`
- **產後課程**: `/img/服務項目/產康圖2.jpg`
- **放鬆課程**: `/img/服務項目/服務項目  需ai重修.jpg`
- **攜伴可約**: `/img/服務項目/服務項目  需ai重修.jpg`
- **預設圖片**: `/img/康妮老師介紹.jpg`

### 如何更新圖片：
1. 將對應的圖片放在 `public/img/` 目錄下
2. 更新 `servicesViews.vue` 中的 `getServiceImage` 函數
3. 可以為每個具體服務設定不同的圖片

## 📱 響應式設計

### 斷點設計：
- **桌面版** (>768px): 多欄網格佈局
- **平板版** (≤768px): 單欄佈局，間距調整
- **手機版** (≤480px): 緊湊佈局，更小間距

### 網格佈局：
```css
grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
```

## 🔄 如何添加新服務

### 1. 更新 JSON 文件：
在 `服務項目.json` 中添加新的服務項目：
```json
{
  "新分類": [
    { "課程名稱": "新服務名稱", "時間": "60min" }
  ]
}
```

### 2. 添加對應圖片：
將圖片放在 `public/img/服務項目/` 目錄下

### 3. 更新圖片映射（可選）：
在 `servicesViews.vue` 中更新 `getServiceImage` 函數
